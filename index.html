Is this using real AI?

This standalone demo uses local example logic so it works without API keys. The correct production setup would use a backend API connected to a model provider.

What would need building next?

Authentication, database storage, backend question generation, spec-grounded retrieval, payment handling, and a proper admin dashboard.

Can it become a real business?

Yes, but only if the questions are genuinely exam-board faithful and the marking is reliable. That is the hard part and the main moat.
